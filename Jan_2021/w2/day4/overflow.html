The way to use overflow property

You can use overflow property when your contents size is larger then parent elements. There are four values following for it.

  1. visible : Default value. Contents will be overflowed when parent elements are smaller than this content.
  2. hidden : Contents won't be overflowed
  3. scroll : Overflowed parts are visible by scrolling
  4. auto : The same as scroll basically, but scroll bar is added only when necessary

Will describe how contents will be appeared when "overflow" is used.

overflow: visible;

When you use "visible" for value of "overflow", it will be like right figure.
You will find that a content is overflowed from parent elements.
