The way to use display property

Display, the property is used to how the elements will be displayed in the web sites. There are mainly four values following for this property.

  1. block : Elements occupy 100% width of parent elements, and it line up vertically
    p, div, ul, h1ã€œh6 etc. tags are block-level elements
  2. inline : Elements line up horizontally
    a, span, img etc. tags are inline elements
  3. inline-block : Elements line up horizontally, and its size is adjustable
  4. none : Elements are removed from layout
