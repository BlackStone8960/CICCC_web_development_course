Box model

How to space between elements?

When we consider about spacing between elements, every elements can be considered as box.

Look at a following figure. When you want to put the picture or the text, it is called as content in HTML. We can configure width and height of content using CSS. Content can be wrapped by the space called padding. And we can set a line called border which goes around content and padding as well. If we want to space between elements, we can space using margin.

You can configure each direction of padding by using padding-top, padding-right, padding-bottom, padding-left which means the direction of top, right, bottom, left respectively.

And also you can code it using shorthand syntax, like padding: 1px 2px 3px 4px;. When you code using this style, the order of directions will be top, right, bottom, left from left parameter to right one. You can also set margin by the same way as padding.

padding-top: 1px;
padding-right: 2px;
padding-bottom: 3px;
padding-left: 4px;
||
padding: 1px 2px 3px 4px;

This is how we can space between elements. The spacing is so important thing when designing web sites, so let's try to style your site using margin or padding. Have a good coding day :)
