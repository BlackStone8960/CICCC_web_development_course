The way to use transform property

Transform property is used when move/rotate/scale/skew elements. There are four functions following for it, and you can use them as the value of this property. Each functions need parameters which mean the direction or angle associated with elements transforming.

  1. translate(translation length along x-axis, translation length along y-axis) : You can move elements along defined values
  2. rotate(angle) : Allows you to rotate elements by defined degrees
  3. scale(ratio of scaling width, ratio of scaling height) : You can scale elements by defined ratio
  4. skew(angle x-axis tilts by, angle x-axis tilts by) : This allows you to skew elements by defined angles

Will describe how contents will be transformed when "transform" is used. I used "transition", and took time interval of 1 sec to make it easy to understand what happened.

transform: translate();

Hover the mouse onto a right square. When you use translate() for value of "transform", it will be like this.
You will find that you can move the elements.

I coded as transform: translate(35px, 35px);, so it moved 35px along each x-axis and y-axis. You can also set just one argument to move along one axis.

transform: scale();

When you use scale(), it will be like this.

I coded as transform: scale(0.75, 0.75);, so its width and height were scaled by 0.75, so it means shrinked by 0.75.

transform: rotate();

When you use translate(), it will be like right square.
You will find that you can rotate the elements.

I coded as transform: rotate(180deg);, so it was rotated by 180 degree.

transform: skew();

At last, what will happen if you use skew()? Hover the mouse onto the square. It will be skewed.

I coded as transform: skew(22.5deg, 22.5deg);, so x-axis and y-axis were tilted by 22.5 degree.
